var: 
 - Объявления всплывают, а присваивания нет
 - Допускается повторное оъявление
 - Функциональная или глобавльная область видимости

Имитация блочной области видимости для var
 (function() {
  var message = "Hello";
  alert(message); // Hello
})();

globalThis - глобавльный объект (window / global)
В браузере глобальные функции и переменные, объявленные с помощью var, становятся свойствами глобального объекта.
То же самое касается функций, объявленных с помощью синтаксиса Function Declaration
